# 흡연 여부 데이터 분석을 통한 건강 인사이트 도출

건강검진 데이터를 활용해 흡연자와 비흡연자의 생체·건강 지표 차이를 분석하고,  
통계적 검정을 통해 흡연이 신체에 미치는 영향을 수치로 검증한 프로젝트입니다.

<br>

## 프로젝트 개요

흡연 여부를 직접 조사하는 것은 현실적으로 어렵습니다. 이 프로젝트는 건강검진 데이터만으로  
흡연자와 비흡연자를 구분 짓는 핵심 생체 지표를 찾아내고, 그 차이를 통계적으로 입증하는 것을 목표로 합니다.  
이후 흡연 여부 예측 모델 개발을 위한 기초 분석 기반 마련에도 활용됩니다.

<br>

## 분석 내용

### 01 | EDA 및 데이터 전처리 (개인)

**전처리**
- BMI 기준 체중 상태 파생 변수 생성 (저체중 / 정상 / 과체중 / 비만)
- 나이 기준 나이대 파생 변수 생성 (청년 / 중년 / 장년 / 노년)
- 결측치 처리: 혈압(중앙값), 시력(최빈값), 중성지방(나이대별 그룹 평균), 공복혈당(전체 평균)

**주요 시각화**
- 흡연 여부별 주요 건강 지표 분포 비교 (Box Plot)
- 나이대별 × 흡연 여부별 혈압·BMI·혈당 변화 추세
- 상관관계 히트맵 (Pearson 상관계수)
- 비만군 흡연자의 간 효소율 분포 (Histogram)
- 흡연/충치 여부에 따른 공복혈당 분포 (Violin Plot)

---

### 02 | 가설 검정 및 이변량 분석 (팀: 무교지만 11조)

**수립 가설 3가지**

> **가설 1.** 흡연 여부에 따라 BMI와 헤모글로빈 수치는 유의미한 차이를 보일 것이다.  
> **가설 2.** 흡연 여부에 따라 동맥경화 지수(AI)는 유의미한 차이를 보일 것이다.  
> **가설 3.** 흡연자 집단 내에서 충치 유무에 따라 대사 지표(공복혈당, 간 효소율)의 차이가 존재할 것이다.

**적용 통계 기법**

| 분석 대상 | 변수 유형 | 사용 기법 |
|-----------|-----------|-----------|
| 흡연 여부 → BMI | 범주 → 수치 | Shapiro-Wilk 정규성 검정 후 Mann-Whitney U Test |
| 흡연 여부 → 헤모글로빈 | 범주 → 수치 | Welch's Independent T-test |
| 흡연 여부 → 동맥경화 지수(AI) | 범주 → 수치 | Mann-Whitney U Test |
| 흡연 여부 ↔ 충치 | 범주 ↔ 범주 | 카이제곱 검정 (Chi-square) |
| 충치 여부 → 공복혈당 / 간 효소율 | 범주 → 수치 | Independent T-test |
| 흡연 그룹별 LDL-콜레스테롤 상관관계 차이 | 수치 → 수치 | Fisher Z-test |

<br>

## 주요 분석 결과

### 상관관계 분석
흡연 여부(label)와 상관관계가 높은 변수 TOP 5

| 순위 | 변수 | 상관계수 |
|------|------|---------|
| 1 | 헤모글로빈 | **0.40** |
| 2 | 중성 지방 | **0.25** |
| 3 | 간 효소율 | ~0.15 |
| 4 | BMI | ~0.10 |
| 5 | 혈압 | **0.02** (나이 분리 시 차이 나타남) |

### 가설 검정 결과

| 가설 | 검정 결과 | 해석 |
|------|-----------|------|
| 가설 1 - BMI | **기각** (p < 0.05) | 흡연자의 BMI가 통계적으로 유의미하게 높음 |
| 가설 1 - 헤모글로빈 | **기각** (p < 0.05) | 흡연자의 헤모글로빈이 유의미하게 높음 — 만성 저산소 보상 반응으로 해석 |
| 가설 2 - 동맥경화 지수(AI) | **기각** (p < 0.05) | 흡연군의 AI가 유의미하게 높음 → 심혈관 질환 위험 증가 |
| 가설 3 - 흡연 ↔ 충치 | **기각** (p < 0.05) | 흡연자 충치 비율 **28.2%** vs 비흡연자 **19.6%** (8.6%p 차이) |
| 가설 3 - 충치 → 공복혈당 | **채택** (p ≥ 0.05) | 충치 유무와 공복혈당 간 유의미한 차이 없음 |
| 가설 3 - 충치 → 간 효소율 | **기각** (p < 0.05) | 충치 있는 그룹에서 간 효소율이 유의미하게 높음 |

### 핵심 인사이트

> **흡연은 단순히 단독 지표를 나쁘게 할 뿐만 아니라, 비만·노화라는 신체적 약점과 결합했을 때 간 수치와 혈압을 폭발적으로 악화시키는 '위험 증폭제' 역할을 한다. 또한 구강 건강과 결합했을 때 대사 질환을 악화시킬 수 있다.**

- 헤모글로빈은 흡연 여부를 가장 잘 설명하는 대표 생리 지표로 확인
- 혈압은 전체 상관관계는 낮지만(r=0.02), 노년층에서 흡연자와 비흡연자 간 격차가 뚜렷하게 확대됨
- 비만(BMI > 30) 흡연자에서 간 효소율 극단적 이상치가 더 빈번하게 발생
